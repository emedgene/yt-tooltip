yt.tooltip

Tooltip directive for angular app

View on GitHub Download .zip Download .tar.gz
Hover{{ $index + 1 }} Hover4

Install

bower install yt-tooltip --save

Usage

angular.module('YourApp', ['yt.tooltip'])
      .controller('myCtrl', ['$scope', function ($scope) {
            $scope.list = [
                  'towry',
                  'jack',
                  'foo'
            ];
      }]);
<div ng-app="YourApp">
  <div ng-controller="myCtrl">
    <a href="#" ng-repeat="item in list track by $index" yt-tooltip tooltip-html="Hello <b>{{ item }}</b>">Hover{{ $index + 1 }}</a>
  </div> 
</div>

Feature

  • Lightweight, for the most simple needs.
  • Browser support: IE9+
  • Only 1.9Kb size
